# SOX2-driven enhancer landscape defines the transcriptional architecture of retinogenesis

This repository is organized by the figures in the publication.

The sequencing data have been deposited in the NCBI Gene Expression Omnibus
(GEO) database under accession number [GSE281033](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE281033), with [GSE281030](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE281030) for ATAC-
seq data, [GSE281031](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE281031) for CUT & RUN seq data and [GSE281032](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE281032) for RNA-seq data.

If you reference any of the scripts in this repository, please cite the original publication:

Fuyun Bian, Kimiasadat Golestaneh, Emily Davis, Abdullah Khan, Brendan Nicholson, Marwa Daghsni, Keevon Flohr, Silvia Liu, Susana da Silva, Len Pennacchio, Issam Aldiri; SOX2-driven enhancer landscape defines the transcriptional architecture of retinogenesis. Development 1 December 2025; 152 (23): dev204986. doi: https://doi.org/10.1242/dev.204986
