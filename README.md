# A SOX2-VSX2 axis modulates a core cis-regulatory circuitry to safeguard retinal neural competence

This repository is broken down by the figures each script contributed to. 

The sequencing data have been deposited in the NCBI Gene Expression Omnibus
(GEO) database under accession number GSE281033, with GSE281030 for ATAC-
seq data, GSE281031 for CUT & RUN seq data and GSE281032 for RNA-seq data.
